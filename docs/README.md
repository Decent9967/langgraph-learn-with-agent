## 💡 学习技巧

### 1. 理论 + 实践

- 先理解概念
- 再看示例代码
- 最后动手实践

### 2. 查阅源码

- 遇到不懂的 API
- 按 F12 查看源码
- 阅读函数签名和文档

### 3. 调试技巧

- 使用 `print()` 打印中间结果
- 查看消息列表
- 分析执行流程

### 4. 笔记记录

- 在讲义上添加自己的理解
- 记录遇到的问题和解决方法
- 总结常用的代码模式
